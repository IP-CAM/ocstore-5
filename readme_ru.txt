------------------------------------------------
OpenCart Модуль Yandex Sitemap
------------------------------------------------
Простая карта сайта в формате XML.
Для поисковых систем не работающих с протоколом  http://www.google.com/schemas/sitemap-image/1.1

Установка
------------
Распакуйте содержимое архива в папку на Вашем компьютере.

Выгрузите содержимое папки "upload" в директорию Вашей установки Opencart.

В админ панели перейдите на страницу Home / Extensions (admin/index.php?route=extension/feed). 
Установите модуль "Yandex Sitemap" нажав на кнопку "Install" напротив модуля.

В админ панели перейдите на страницу Home / Extensions / Feeds Sitemap (admin/index.php?route=extension/feed/yandex_sitemap).
Включите модуль выбрав пункт "enabled" в выпадающем меню и нажмите кнопку "Save".

Для формирования карты сайта перейдите по ссылке:
http://your-domain-name/index.php?route=extension/feed/yandex_sitemap
